title POST Request Diagram

browser->server: HTTP POST https://studies.cs.helsinki.fi/exampleapp/new_note \n Payload { note: String }
note over server:
server gets the POST request and appends the string passed 
on with the payload to the data.json file
end note
server-->browser: Response Code 302 (Redirect to ...exampleapp/notes)

note over browser:
browser reloads the webpage
end note

browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/notes
server-->browser: HTML-code
browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.css
server-->browser: main.css
browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.js
server-->browser: main.js
browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/data.json
